# AI-Learn-Go (使用AI学习Golang)

# Go 语言学习指南

本目录包含一系列 Go 语言学习示例文件，按照由浅入深的顺序组织。每个文件都可以独立编译运行。

## 运行方式

```bash
# 运行单个文件
go run 01_hello_world.go

# 或者构建后运行
go build 01_hello_world.go
./01_hello_world
```

## 学习建议

### 初学者路线（1-2周）
1. 先学习 01-07，掌握基础语法和函数
2. 然后学习 08-09，掌握数据结构
3. 最后学习 05-06 的高级用法

### 进阶路线（2-3周）
1. 学习 10-12，掌握面向对象编程
2. 学习 13-14，理解指针和错误处理
3. 学习 17，理解 defer 机制

### 高级路线（3-4周）
1. 学习 15-16，掌握并发编程
2. 学习 19，掌握同步原语
3. 学习 20-22，掌握泛型、JSON 和 Context

## 重要概念

### Go 的设计哲学
- **简洁**：语法简单，易于学习
- **高效**：编译快，运行快
- **并发**：原生支持并发编程
- **实用**：工具链完善

### 核心特性
- 静态类型，编译型语言
- 垃圾回收
- 内置并发（goroutine + channel）
- 接口隐式实现
- 函数是一等公民

### 常用命令
```bash
go run main.go      # 编译并运行
go build            # 编译
go test             # 运行测试
go fmt              # 格式化代码
go mod init         # 初始化模块
go get              # 下载依赖
go doc              # 查看文档
```
